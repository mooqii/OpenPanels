# File deduplication

Pixcall scans its asset library in real time for duplicate files. It detects exact duplicates across all file types and can also detect identical or visually similar ordinary image files.

## Resolving duplicates

For a duplicate group, Pixcall automatically selects the preferred file, filename, and file path to retain. The user can override those choices manually.

Duplicate handling supports:

- Merging one duplicate group.
- Merging every group in the current list, up to 100 groups per list.
- Ignoring one duplicate group.
- Ignoring every group in the current list, up to 100 groups per list.
- Excluding an individual file from the duplicate-file list.

---
name: writing-default
description: Follow the user's writing request directly without imposing an additional house style.
---

Follow the user's writing instruction directly.

1. Treat the user's instruction as the primary creative brief.
2. Use supplied context as source material. Do not invent facts, quotations,
   sources, experiences, or claims.
3. Match the requested language. When none is specified, use the language of
   the user's instruction.
4. Preserve explicit requirements for audience, purpose, tone, structure,
   length, and format.
5. Do not impose a house style, platform convention, rhetorical device, or
   extra section that the user did not request.
6. Deliver only the requested writing unless brief supporting context is
   necessary to make the result usable.

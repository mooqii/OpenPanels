description: 围绕文章的核心主题与情绪，创作清晰、有辨识度的编辑封面。
---

默认根据提供的文章创作一张编辑封面图。如果请求明确要求多个选项，则为每个选项分别创作一张完整封面图。

1. 结合标题与正文阅读，提取核心主题、情绪基调和一到两个具体视觉意象。不要只图解开头第一句话。
2. 遵循用户对主体和风格的明确要求，但不要加入文章未支持的事实表达。
3. 采用接近 4:3 的横向构图，设置单一有力的视觉焦点、克制的留白、清楚的层级，以及缩小为缩略图后仍然有效的对比度。
4. 优先使用贴合主题的编辑视觉，避免通用图库摆拍、装饰性渐变或与内容无关的氛围画面。
5. 除非用户明确要求排印文字，否则不要生成文字、字母、标志、水印、界面元素或仿制的出版标识。
6. 只产出完成度充分的 PNG 位图封面，不要返回联系表、样机或解释。

整理提供的文章，使其具备清楚、自然的编辑阅读结构。

1. 保留所有文字及其原始顺序。不要改写、纠错、翻译、总结、增加或删除文字。
2. 保留所有链接与图片，包括目标、内容和相对位置。
3. 只有在结构明确时，才规范段落边界与空白。
4. 只把原本就承担标题作用的文字设置为标题，并建立连贯的层级；不要把普通正文提升为标题。
5. 仅当原文已经表达列表或引用结构时，使用列表或块引用。
6. 谨慎使用粗体和斜体来保留或澄清既有强调，不要添加会改变文章含义的装饰样式。
7. 只返回调用方要求的结构化文档。完成前确认文字、链接与图片均无损保留。

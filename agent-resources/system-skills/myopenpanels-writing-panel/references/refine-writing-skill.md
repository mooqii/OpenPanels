Use this reference only inside the exact claimed `refine_writing_skill` Task
handoff.

1. Read the captured refinement request and selected example documents.
2. Load the task-selected refinement Skill and follow its content method.
3. Produce one portable `SKILL.md` package without MyOpenPanels commands,

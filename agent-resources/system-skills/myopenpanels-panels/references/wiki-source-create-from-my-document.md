# Create A Wiki Source From A My Document

Use this reference only when the user explicitly asks to publish a My Document
into the Wiki raw-source layer.

1. Resolve and read exactly one My Document.
2. Resolve the destination Wiki space without changing the visible panel.
3. Run the advertised high-risk `wiki-source.create-from-my-document` command once.
4. Report the resulting raw document and any downstream Task state.

Publication is distinct from creation and revision. Never publish merely
because a My Document was completed.

# Read Writing Context

Use this reference when the user asks what source material is selected for a
Writing request or asks to use that material outside a claimed Writing Task.

1. Require the Writing panel to be active and read its current selection.
2. Treat selected raw and My Documents as direct references.
3. Treat a selected Wiki as optional background knowledge; search it and read
   only relevant pages.
4. Use verified local paths only for oversized content or file-oriented tools.
5. Keep this Procedure read-only. Submitted Writing requests are executed only
   through their exact Task handoff.

# Delete A My Document

Use this reference only for an explicit request to delete a My Document.

1. Resolve exactly one target by explicit selection, id, or an unambiguous
   title, and read it before deletion.
2. Do not substitute a Wiki raw document or generated Wiki page.
3. Run the advertised high-risk `my-document.delete` command once.
4. Report the deleted document id. Stop when the target changed, disappeared,
   or remained ambiguous.

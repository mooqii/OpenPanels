# Delete A Generated Wiki Document

Use this reference only for an explicit request to delete a standalone
generated document.

1. Resolve exactly one target by explicit selection, id, or an unambiguous
   title, and read it before deletion.
2. Do not substitute a raw document or generated Wiki page.
3. Run the advertised high-risk `wiki.document.delete` command once.
4. Report the deleted document id. Stop when the target changed, disappeared,
   or remained ambiguous.
